Lesson 33: Mouse Hover Events in JavaScript

JavaScript lets you detect mouse interactions with several events:

  • mouseover: when mouse enters an element or its child
  • mouseout: when mouse leaves an element or its child
  • mouseenter: when mouse enters an element (no bubbling)
  • mouseleave: when mouse leaves an element

Example:

document.querySelector(".hover-box").addEventListener("mouseenter", () => {
  console.log("Mouse entered the box!");
});

document.querySelector(".hover-box").addEventListener("mouseleave", () => {
  console.log("Mouse left the box!");
});
