Lesson 28: Fetch API & External Data

The Fetch API is a modern way to make HTTP requests. It's commonly used to retrieve data from external APIs and process JSON responses.

Example: Fetch JSON Data


fetch('https://jsonplaceholder.typicode.com/users') .then(response => response.json()) .then(data => { console.log(data[0].name); // Shows the name of first user });
