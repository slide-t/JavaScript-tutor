Lesson 3: JavaScript Data Types & Operations

JavaScript supports different data types including:

  • String
  • Number
  • Boolean
  • Undefined
  • Null
  • Object

Example 1: Strings and Numbers

let name = 'Azeez'; let age = 12; console.log('Name: ' + name); console.log('Age: ' + age);

Example 2: Boolean and Null

let isEnrolled = true; let graduationDate = null; console.log(isEnrolled); console.log(graduationDate);
