What is sessionStorage?

sessionStorage stores data that is only available during a page session (until the browser is closed).

sessionStorage.setItem("theme", "dark");
let theme = sessionStorage.getItem("theme");
console.log(theme); // Output: dark

localStorage vs sessionStorage

  • localStorage: Persists even after the browser is closed.
  • sessionStorage: Clears automatically when the page session ends.
