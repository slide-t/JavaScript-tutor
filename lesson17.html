Lesson 17: addEventListener & Event Object

addEventListener() lets you attach events to elements without inline HTML. You can also access event details with the event object.

Example 1: Using addEventListener

document.getElementById("myBtn").addEventListener("click", function() { alert("Clicked!"); });
