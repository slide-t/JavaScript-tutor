üß† Function Expressions

Functions can be stored in variables. These are called function expressions.

const greet = function(name) {
  return "Hi " + name + "!";
};

console.log(greet("Ali"));
// Output: Hi Ali!

‚ö° Arrow Functions (ES6+)

Arrow functions are a modern, shorter way to write functions.

const greet = (name) => {
  return "Hello " + name;
};

console.log(greet("Amina")); // Hello Amina

ü™∂ One-liner Arrow Functions

If there's just one return statement, you can omit return and curly braces.

const add = (a, b) => a + b;

console.log(add(2, 3)); // 5
‚úÖ Arrow functions do not bind their own this ‚Äî useful for callbacks and concise logic.
