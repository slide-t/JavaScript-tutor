üõë What is break?

The break statement exits a loop immediately.

for (let i = 1; i <= 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
// Output: 1 2 3 4

‚è≠Ô∏è What is continue?

continue skips just the current loop iteration and goes to the next.

for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    continue;
  }
  console.log(i);
}
// Output: 1 2 4 5
‚úÖ Use continue when you want to skip a value (e.g., even numbers, nulls, etc.)
