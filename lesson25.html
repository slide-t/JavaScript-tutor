Lesson 25: JavaScript Timers

JavaScript provides two main timer functions: setTimeout and setInterval. These are used to delay or repeat actions in time.

Example 1: setTimeout


setTimeout(function() { alert("This message shows after 3 seconds"); }, 3000);

Example 2: setInterval


setInterval(function() { console.log("This logs every 2 seconds"); }, 2000);
