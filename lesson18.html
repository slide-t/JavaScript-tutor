üß† Lexical Environment

A function remembers the variables from its parent scope:

function outer() {
  let name = "Ali";

  function inner() {
    console.log("Hello " + name); // remembers 'name'
  }

  return inner;
}

const greet = outer();
greet(); // Hello Ali
‚úÖ Use let and const for block-scoped variables to avoid bugs.
