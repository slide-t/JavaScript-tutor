Lesson 12: JavaScript Objects

Objects in JavaScript are used to store multiple values in a single variable using key-value pairs.

Example 1: Creating an Object

let person = { name: "Azeez", age: 10, school: "EduKids Academy" }; console.log(person.name); // Azeez

Example 2: Adding and Changing Properties

person.age = 11; person.country = "Nigeria"; console.log(person);
