üß≠ map()

Transforms every element in an array and returns a new array.

const nums = [1, 2, 3, 4];
const doubled = nums.map(n => n * 2);
console.log(doubled); // [2, 4, 6, 8]

üßπ filter()

Returns a new array with only elements that pass a condition.

const scores = [40, 85, 60, 95];
const passed = scores.filter(score => score >= 60);
console.log(passed); // [85, 60, 95]

üßÆ reduce()

Combines all array elements into a single value.

const prices = [10, 20, 15];
const total = prices.reduce((sum, price) => sum + price, 0);
console.log(total); // 45
‚úÖ map returns same length array. filter may return fewer items. reduce always returns one value.
üí¨ Interview Tip: You may be asked to chain these methods or rewrite loops using map/filter/reduce.
