Lesson 27: JSON in JavaScript

JSON (JavaScript Object Notation) is a lightweight format for storing and transporting data, often used when exchanging data between a web server and client.

Example 1: Convert Object to JSON


const user = { name: "Alice", age: 30, isMember: true };

const jsonStr = JSON.stringify(user); console.log(jsonStr);

Example 2: Convert JSON to Object


const data = '{"name":"Alice","age":30,"isMember":true}'; const obj = JSON.parse(data); console.log(obj.name); // Alice
