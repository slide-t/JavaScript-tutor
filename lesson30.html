Lesson 30: ES6 Features Overview

ES6 (ECMAScript 2015) introduced many new and powerful features to JavaScript. Here's a summary of the most important ones.

1. let and const


let name = "John"; const age = 30;

2. Arrow Functions


const greet = () => console.log("Hello!");

3. Template Literals


const name = "Jane"; console.log(Welcome, ${name}!);

4. Destructuring


const user = { name: "Tom", age: 25 }; const { name, age } = user;

5. Default Parameters


function sayHi(name = "Guest") { console.log("Hi " + name); }
