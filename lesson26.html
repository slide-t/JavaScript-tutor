Lesson 26: Working with Local Storage

Local storage lets you store data in the browser persistently across sessions. It uses key-value pairs and has no expiration time until explicitly cleared.

Example 1: Save to Local Storage


localStorage.setItem("username", "JohnDoe");

Example 2: Retrieve from Local Storage


let user = localStorage.getItem("username"); console.log(user); // JohnDoe

Example 3: Remove from Local Storage


localStorage.removeItem("username");
