Lesson 36: Form Validation with JavaScript

Form validation helps ensure that users provide necessary and properly formatted information before submitting a form.

1. Simple Required Validation


form.addEventListener("submit", function(e) { const username = document.getElementById("username"); if (username.value.trim() === "") { e.preventDefault(); alert("Username is required"); } });

2. Pattern and Length Validation


const email = document.getElementById("email"); const emailPattern = /^[^\s@]+@[^\s@]+.[^\s@]+$/; if (!emailPattern.test(email.value)) { e.preventDefault(); alert("Enter a valid email"); }
