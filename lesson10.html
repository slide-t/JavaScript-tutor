Lesson 10 ‚Äì Ternary Operator

üéØ What is the Ternary Operator?

The ternary operator is a shorthand way to write if...else statements. It uses the syntax:

condition ? expressionIfTrue : expressionIfFalse

üß™ Example: Age Check

let age = 20;

let message = (age >= 18) ? "You can vote!" : "Too young to vote.";

console.log(message);

üì¶ Cleaner Alternative to if...else

let isLoggedIn = true;

let welcome = isLoggedIn ? "Welcome back!" : "Please log in.";
console.log(welcome);
‚úÖ Use ternary when assigning a value based on a condition, especially inside let or const.
‚ö†Ô∏è Don't overuse ternaries with many nested conditions ‚Äî it reduces readability.

üí° Nesting (Be Careful!):

let score = 85;

let grade = (score >= 90) ? "A" :
            (score >= 80) ? "B" :
            (score >= 70) ? "C" : "F";

console.log("Your grade is:", grade);
