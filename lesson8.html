Lesson 8 ‚Äì Conditional Statements

üß† Why Use Conditionals?

JavaScript conditionals allow your code to make decisions and behave differently depending on inputs or variables.

üìå Syntax

if (condition) {
  // code runs if condition is true
} else if (anotherCondition) {
  // code runs if anotherCondition is true
} else {
  // code runs if none are true
}

üéØ Real-World Example: Age Check

let age = 17;

if (age >= 18) {
  console.log("You can vote!");
} else {
  console.log("Sorry, too young to vote.");
}

üéØ Grading System Example

let score = 75;

if (score >= 90) {
  console.log("A");
} else if (score >= 80) {
  console.log("B");
} else if (score >= 70) {
  console.log("C");
} else {
  console.log("Try again.");
}

üì¶ Dynamic Use: Simple Login

let username = "admin";
let password = "pass123";

if (username === "admin" && password === "pass123") {
  alert("Welcome Admin!");
} else {
  alert("Invalid credentials.");
}
