Lesson 9: JavaScript Events

Events let you respond to user interactions like clicking, typing, or moving the mouse.

Example 1: Click Event

function sayHello() { alert('Hello from EduKids Africa!'); }

Example 2: Using addEventListener

let btn = document.getElementById('myBtn'); btn.addEventListener('click', function() { alert('Button clicked!'); });
