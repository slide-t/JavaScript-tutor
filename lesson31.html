Lesson 31: DOM Manipulation Overview

The Document Object Model (DOM) represents the structure of a webpage and allows JavaScript to interact with and manipulate it.

1. Selecting Elements


const title = document.getElementById("title"); const buttons = document.querySelectorAll("button");

2. Changing Content


document.getElementById("demo").textContent = "Hello DOM!";

3. Changing Styles


document.querySelector(".box").style.backgroundColor = "blue";

4. Adding Event Listeners


document.querySelector("button").addEventListener("click", function() { alert("Button clicked!"); });
