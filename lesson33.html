Lesson 33: Creating and Removing Elements

In JavaScript, you can dynamically create and remove elements from the DOM using methods like createElement(), appendChild(), and remove().

1. Creating Elements


const newDiv = document.createElement("div"); newDiv.textContent = "Hello from JS!"; document.body.appendChild(newDiv);

2. Removing Elements


const elementToRemove = document.getElementById("oldItem"); elementToRemove.remove();
