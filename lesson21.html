Lesson 21: JavaScript Classes & Constructors

Classes are templates for creating objects. They help organize and reuse code using constructors and methods.

Example 1: Creating a Class


class Person { constructor(name, age) { this.name = name; this.age = age; }

greet() { return Hello, my name is ${this.name} and I'm ${this.age} years old.; } }

const john = new Person("John", 25); console.log(john.greet());
