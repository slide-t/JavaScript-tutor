Lesson 5: JavaScript If Statements

If statements are used to run code only if a certain condition is true.

Example 1: Basic If Statement

let score = 85; if (score > 80) { console.log('Great job!'); }

Example 2: If-Else Statement

let age = 15; if (age >= 18) { console.log('You are an adult.'); } else { console.log('You are a minor.'); }
