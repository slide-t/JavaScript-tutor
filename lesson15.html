Lesson 15: Array Iteration (forEach & map)

JavaScript provides powerful methods to loop through arrays: forEach() and map().

Example 1: Using forEach()

let colors = ["Red", "Green", "Blue"]; colors.forEach(function(color) { console.log(color); });

Example 2: Using map()

let numbers = [1, 2, 3]; let doubled = numbers.map(function(num) { return num * 2; }); console.log(doubled);
